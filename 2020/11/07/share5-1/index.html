JavaScript 对象是被命名值的容器。

对象简介

对象属性: 变量的键值对
对象也可以有方法。
方法是在对象上执行的动作。
方法以函数定义被存储在属性中。
方法是作为属性来存储的函数。

对象访问

对象可以通过两种方式访问, 一种是通过 . 操作符访问 如: object.property 或者 object.method("method"), 第二种方式就是通过 [] 访问 如:object[property] 或者 object[method](method)

这两种的区别在于,使用中括号的方式可以在中括号中使用变量或者表达式 例如 object["me"+"th"+"od"]()

对象创建

在javascript中对象可以方便的用 JSON 格式来描述一个对象, 只需要使用他就可以直接创建对象,上文中的 object就是一个例子,另外的, 和一般的面向对象一样, 我们可以通 new 操作符来完成对象的创建. 这里我们就需要提到一种叫做构造函数的东西. 构造函数和一般的函数没有区别, 任何非箭头函数都可以作为对象的构造函数.

除此之外,我们也可以用 Object.create()创建拥有特定原型的对象

对象的属性

对象属性的权限问题.

特性名称 |描述| 默认值
| —- | —- | —- |
value| 属性的值 |undfined
writable| 是否可以修改属性的值，true表示可以，false表示不可以 |true
enumerable| 属性值是否可枚举，true表示可枚举for-in, false表示不可枚举 |true
configurable |属性的特性是否可配置，表示能否通过delete删除属性后重新定义属性|true

这些属性不能直接定义,需要通过Object.defineProperty()来定义

使用 in 操作符可以 判断对象中是否存在某属性, 例如

a 中 虽然没有 toString 但是 返回值仍然是 true

虽然 a 中没有该属性, 但是他的原型中有.

单单使用 json 的方式创建的并不是一个空的对象, 即使被赋值为 {} 里面也会有内置的属性, 所有的对象都会继承自js内部的对象, 当访问对象中不存在的属性时, 对象会根据原型链去原型中去寻找, 一层层向上寻找,直到达到最顶端.这就形成了原型链. 空对象的原型是什么呢?是Object.prototype,它是所有对象中原型的父原型.
除去 自定义的属性, 对象会被加入一个 __protp__ 属性, 通过一般构造函数创建的对象,其__protp__中有两个属性,一个是constructor,另一个是__protp__,前一个指向他的构造函数, 第二个指向上一层原型.

先将__protp__ 放在一边, 如何只检测自己的属性呢, Object.hasOwnProperty()，判断一个对象是否有指定名称的属性，不会检查继承属性,Object.propertyIsEnumerable()，判断指定名称的属性是否为实例属性并且是可枚举的, Object.keys()返回一个数组，内容是对象可枚举的实例属性名称

Object中有许多的方法,可以通过浏览器或者IDE查看.

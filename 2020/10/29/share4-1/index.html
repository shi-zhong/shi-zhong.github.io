使用

js-xlsx 提供了一个XLS 和 XLSX对象给我们;在其内部定义了一个wb(workbook)对象,用来记录和描述一个xlsx等表格文件的内容.在里面有两个属性存储表格的文本数据: SheetNames 和 Sheets

Sheet Object

主要看Sheet里面的属性 字母数字的组合代表了一个普通单元格,!开头的对象则记录了表格的特殊信息,例如 !ref记录了sheet表格的范围,!merges记录了单元格的合并信息,还有!rows/!cols…

单元格对象

每一个单元格是一个对象（Cell Object），主要有t、v、r、h、w等字段（详见这里）：

v：表示原始值；
w：格式化后的内容;
t：表示内容类型，s:string，n:nmber，b:boolean，d:date, e:error, z:blank stub..
f：表示公式，如B2+B3；
h：HTML内容;
r：富文本内容rich text;
s: 单元格样式或主题(if applicable)

XLSX 对象

xlsx对象提供给我们了一系列的函数和工具主要的就是utils, read 和 write, utils提供了一系列的工具函数来帮助我们进行数据的转换,readFile/writeFile…都是在读写两个函数的基础上进行了包装. 在官方文档中对很多内容都进行了解释.

utils

里面的函数大致分为 四 类:编码解码/格式转换/创建新对象/设置内容;
Some helper functions in XLSX.utils generate different views of the sheets:

  • XLSX.utils.sheet_to_csv generates CSV
  • XLSX.utils.sheet_to_txt generates UTF16 Formatted Text
  • XLSX.utils.sheet_to_html generates HTML
  • XLSX.utils.sheet_to_json generates an array of objects
  • XLSX.utils.table_to_book generates workbook object
  • …

book 指的是 workbook 工作对象, 里面的sheet指的就是book中的一张表可以通过XLSX.utils.book_append_sheet将sheet插入到book中,对象转换类的函数都可以顾名思义,通过名自判断出对应的效果.

Parsing functions

XLSX.read(data, read_opts) attempts to parse data.
XLSX.readFile(filename, read_opts) attempts to read filename and parse.

Writing functions

XLSX.write(wb, write_opts) attempts to write the workbook wb
XLSX.writeFile(wb, filename, write_opts) attempts to write wb to filename.
XLSX.writeFileAsync(filename, wb, o, cb) attempts to write wb to filename.
